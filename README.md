# typoramark

技术栈: 该项目采用后端采用marked 前端采用SimpleMDE

## 使用说明

若需要更加良好的体验 需要安装字体 fira code 和 仿宋

点击右下角的二刺猿头像即可打开设置抽屉.
对应的功能不作多余讲解

### 使用编辑

选择文件之后 按住 唱跳rap篮球 + / 进入编辑模式
当你编辑完成之后 按住  唱跳rap篮球 + s  保存文件 保存文件后 再次按住 唱跳rap篮球 + / 退出编辑模式
